Setup Instructions

1 Configure Breeze API
  • Run setup script: python scripts/setup_auto_login.py
  • Enter your ICICI Direct User ID and Password when prompted
  • Optionally add TOTP secret for automatic OTP generation
  • Click "Manual Login" button above to test the connection
2 Configure Kite Connect
  • Ensure API key and secret are configured in .env file
  • Run setup script for Kite credentials
  • Enter your Zerodha User ID, Password, and PIN
  • Test connection using the button above
3 Schedule Automatic Login
  • Go to the Scheduler Dashboard
  • Configure login times (recommended: 5:30 AM and 8:30 AM)
  • Enable weekdays-only option for trading days
  • Start the scheduler to enable automatic daily login
Manual Login Commands:

• Breeze: python scripts/auth/breeze_auto_login.py
• Kite: python scripts/auth/kite_auto_login.py
• Setup: python scripts/setup_auto_login.py

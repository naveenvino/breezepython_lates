Breeze (ICICI) Setup
1
Login to ICICI Direct
Go to secure.icicidirect.com and login with your credentials
2
Navigate to API Section
Go to Customer Service → API → Breeze API
3
Generate Session
Click "Generate Session" and get your 8-digit token
4
Update Credentials
Enter token, API key, and secret in the Breeze section above
